## Fonctionnalités

- **Messagerie instantanée** — Discutez en temps réel avec vos amis
- **Appels vocaux** — Passez des appels audio haute qualité
- **Partage d'écran** — Partagez votre écran en un clic
- **Serveurs & salons** — Créez vos communautés
- **Messages privés** — Conversations privées et groupes
- **Mises à jour automatiques** — L'application se met à jour automatiquement

## Configuration requise

- **Windows** : Windows 10 ou supérieur
- **macOS** : macOS 10.13 (High Sierra) ou supérieur
- **Linux** : Ubuntu 18.04, Fedora 32, ou équivalent

## Installation

### Windows
1. Téléchargez le fichier `.exe` depuis les [releases](https://github.com/ouinsurinternet/whoops-app/releases/latest)
2. Lancez l'installateur
3. Whoops se lance automatiquement

### macOS
1. Téléchargez le fichier `.dmg` depuis les [releases](https://github.com/ouinsurinternet/whoops-app/releases/latest)
2. Ouvrez le DMG et glissez Whoops dans Applications
3. Lancez Whoops depuis le Launchpad

### Linux
1. Téléchargez le fichier `.AppImage` depuis les [releases](https://github.com/ouinsurinternet/whoops-app/releases/latest)
2. Rendez-le exécutable : `chmod +x Whoops.AppImage`
3. Lancez-le : `./Whoops.AppImage`

## Développement

```bash
# Cloner le repo
git clone https://github.com/ouinsurinternet/whoops-app.git
cd whoops-app

# Installer les dépendances
npm install

# Lancer en mode développement
npm start

# Build
npm run build:win    # Windows
npm run build:mac    # macOS
npm run build:linux  # Linux
```
